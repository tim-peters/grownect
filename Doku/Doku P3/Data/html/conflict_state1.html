INITIALISIERUNG

Der User darf sich nun anmelden, ein Profilbild hochladen (mittels QR-Code Upload), seinen Namen eingeben, seine Farbe auswählen - die dann auch auf dem Home-Screen für Andere User zur Wiedererkennung gut sind. Ebenfalls dürfen sie eine Beschreibung über sich selbst eingeben.

Hier wird jeder User angelegt der involviert ist. Die Anzahl hierfür steht vor der Produktvermietung fest.
