ADD CONFLICT

Ist der Ärger durch das Schreien nicht verflogen Muss der User nun selbst einen Konflikt anlegen, indem er auf den Konfliktpartner klickt. Der User selbst und der Konfliktpartner werden auf dem Screen zur verdeutlichung der startenden Interaktion größer angezeigt.

Der Puls wird ein zweites Mal gemessen. Ist Dieser zu hoch wird der Nutzer aufgefordert ein zweites Mal zu schreien. Ansonsten bekommt er weitere Varianten zur Verfügung gestellt.
