• Nutzer-Erkennung

  • Automatische Erkennung des Spiegels durch passiver RFID-Tag in Kommunikation zu Armband das einen aktiven RFID-Reader bestitz. Dahinter läuft ein Server der Daten speichert und diese gezielt anrufgerufen wird mittels Datenbank. Aufgrund der Erkennung kann er aus der Datenbank Informationen dazu abrufen und der Server kann im Hintergrund die Daten berechnen die er individuell angezeigt bekommt. (ID-Erkennung)
